1. LIST

What Is A List?

A List is a collection that stores multiple values inside a single variable.

Example

Students

↓

Sohan

Rahul

Aman

Ravi

↓

Stored Inside One List

Why Is It Important?

Because large amounts of data can be stored efficiently.

Features

✓ Ordered

✓ Mutable

✓ Duplicate Values Allowed

✓ Uses Square Brackets [ ]

Real-Life Uses

✓ Student Records

✓ Product Lists

✓ Sensor Readings

✓ Employee Names

✓ Marks Storage

9. TUPLE

What Is A Tuple?

A Tuple is a collection that stores multiple values but cannot be modified.

Why Do We Need It?

Some data should remain fixed.

Examples

Date Of Birth

Coordinates

Machine Configuration

System Settings

Features

✓ Ordered

✓ Immutable

✓ Faster Than Lists

✓ Uses Parentheses ( )

🎯 MOST IMPORTANT INTERVIEW QUESTIONS

Q. What is a List?

Answer

A mutable collection used to store multiple values.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q. What is a Tuple?

Answer

An immutable collection used to store fixed values.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q. What is a Dictionary?

Answer

A collection that stores data using Key : Value pairs.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q. Difference Between List And Tuple?

Answer

List = Mutable

Tuple = Immutable

Q. Which collection uses Key : Value pairs?

Answer

Dictionary

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q. What does append() do?

Answer

Adds an element at the end of a list.
