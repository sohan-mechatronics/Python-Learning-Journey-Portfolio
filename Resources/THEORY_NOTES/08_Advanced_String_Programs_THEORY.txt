1. PALINDROME STRING ⭐

What Is A Palindrome?

A Palindrome is a word that remains the same when read forward and backward.

Examples

MADAM

LEVEL

RADAR

MALAYALAM

Why Is It Important?

Because it is one of the most common interview and exam questions.

Concepts Learned

✓ String Reversal

✓ Comparison

✓ Logic Building

Logic

Example

Reverse = MADAM

Result = Palindrome

Golden Rule
