What Is A For Loop?

A For Loop repeats code a fixed number of times.

Why Is It Important?

Because many tasks require known repetitions.

Examples

Print Tables

Process Data

Generate Reports

✓ Better Readability

2. RANGE FUNCTION

What Is range()?

range() generates a sequence of numbers automatically.

Why Do We Need It?

For Loops need values to iterate through.

Important Rule

Stop Value

↓

Not Included

Example

↓

1 To 10

Benefits

✓ Automatic Number Generation

✓ Cleaner Loops

✓ Easy Counting

Q. What is a For Loop?

Answer

A loop used for fixed repetitions.

Q. What is range()?

Answer

A function that generates a sequence of numbers.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q. Is the stop value included in range()?

Answer

No.

The stop value is excluded.

Answer

Answer

Multiplication of all positive integers up to a number.
