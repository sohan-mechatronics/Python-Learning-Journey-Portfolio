Meaning

pip = Package Installer

numpy = Numerical Library

Result

NumPy can perform operations on the entire array at once.

Example Array

[10 20 30 40 50]

[20 30 40 50 60]

Multiplication

Division

[1. 2. 3. 4. 5.]

A Matrix is a 2D Array consisting of rows and columns.

Syntax

[10,20,30],
[40,50,60]
])

Output

[[10 20 30]
 [40 50 60]]

14. RESHAPE

Why?

Sometimes data must be reorganized into a new structure.

Original Array

[10 20 30 40 50 60]

Syntax

numbers.reshape(2,3)

Output

[[10 20 30]
 [40 50 60]]

Rule

Total Elements Must Remain Same

Valid

reshape(2,3)

reshape(3,2)

reshape(1,6)

reshape(6,1)

Benefits

✓ Data Transformation

✓ AI & ML Data Preparation

✓ Matrix Conversion
