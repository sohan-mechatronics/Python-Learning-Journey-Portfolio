2. CURRENT DATE & TIME

Problem

How can we know the current date and time?

Solution

datetime.datetime.now()

Example

import datetime

current = datetime.datetime.now()

print(current)

Output

2026-06-01 18:35:20

Understanding Output

2026 = Year

06 = Month

01 = Day

18 = Hour

35 = Minute

20 = Second

Benefits

✓ Real-Time Information

✓ Event Tracking

✓ Logging Systems

5. DATE FORMATTING

Problem

Default date format may not be user-friendly.

Example

2026-06-01

↓

Need

01-06-2026

Solution

strftime()

Definition

strftime() converts date and time into custom formats.

Example

today.strftime("%d-%m-%Y")

Output

01-06-2026

Why Important?

Used in:

✓ Reports

✓ Certificates

✓ Dashboards

✓ Attendance Systems

✓ Professional Applications

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

6. MOST IMPORTANT FORMAT CODES

%d = Day

%m = Month

%Y = Full Year

%H = 24-Hour Format

%I = 12-Hour Format

%M = Minute

%S = Second

%p = AM / PM

%A = Day Name

Example

strftime("%d-%m-%Y")

Output

01-06-2026

Example

strftime("%H:%M:%S")

Output

21:45:30

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
