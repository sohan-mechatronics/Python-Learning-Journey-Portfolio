1. FUNCTIONS

What Is A Function?

A Function is a reusable block of code that performs a specific task.

Think Of It Like

Input

↓

Processing

↓

Output

Example

greet()

↓

Displays Greeting Message

Why Is It Important?

Because functions reduce repetition and improve program organization.

Benefits

✓ Reusable Code

✓ Cleaner Programs

✓ Easy Maintenance

✓ Professional Structure

Definition

The return statement sends a value back from a function.

Benefits

✓ Reusable Results

✓ Better Logic

✓ Professional Programming

Important Difference

↓

Displays Result

↓

9. PRINT VS RETURN

print()

Purpose

Show output on screen.

Use When

Result only needs to be displayed.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

return

Purpose

Send value back to the program.

Use When

Result needs to be stored, reused or processed further.

Golden Rule

print()

= Show

return

= Send Back
