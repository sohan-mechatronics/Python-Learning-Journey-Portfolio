⚠ COMMON MISTAKES & LESSONS

Problem 1

Forgot:

import numpy as np

Result

NameError

Lesson

Always import NumPy first.

────────────────────────

Problem 2

Wrong Reshape

6 Elements

reshape(4,4)

Result

Error

Lesson

Total elements must remain same.

────────────────────────

Problem 3

Using Python List Functions Instead Of NumPy

Wrong

numbers.mean()

(on normal list)

Lesson

Use NumPy arrays for NumPy functions.

────────────────────────

Problem 4

Confusing Shape and Size

shape

Rows × Columns

size

Total Elements

Lesson

Never mix them.
