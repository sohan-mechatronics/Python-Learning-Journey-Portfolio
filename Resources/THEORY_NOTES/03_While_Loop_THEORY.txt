1. WHILE LOOP

What Is A While Loop?

A While Loop repeats a block of code as long as a condition remains True.

Why Is It Important?

Because many real-world tasks require repetition.

Examples

Print Numbers

Check Passwords

Process Data

Run Machines

Monitor Sensors

Benefits

✓ Automation

✓ Less Code

✓ Repetition Control

✓ Real-World Applications

5. INFINITE LOOP

What Is An Infinite Loop?

A loop that never stops executing.

Main Reason

Missing Update Statement

Example

count = count + 1

not written

↓

Condition never changes

↓

Loop never ends

Why Is It Dangerous?

✓ Program Hangs

✓ CPU Usage Increases

✓ User Cannot Continue

Important Lesson

Every While Loop needs proper updating.

🎯 MOST IMPORTANT INTERVIEW QUESTIONS

Q. What is a While Loop?

Answer

A loop that repeats code while a condition remains True.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q. What is a Counter Variable?

Answer

A variable used to control loop execution.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q. What is Increment?

Answer

Increasing a variable value.

Example

count = count + 1

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q. What is Decrement?

Answer

Decreasing a variable value.

Example

count = count - 1

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q. What causes an Infinite Loop?

Answer

Missing update statement or a condition that never becomes False.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q. What is an Accumulator Variable?

Answer

A variable used to collect and store totals during loop execution.
