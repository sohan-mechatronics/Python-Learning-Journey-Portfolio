2. IMPORT STATEMENT

What Is Import?

The import keyword loads a module into the current program.

Syntax

import math

Meaning

Python,

load the Math Module into my program.

Example

import math

print("Module Imported Successfully")

Output

Module Imported Successfully

Benefits

✓ Access Ready-Made Functions

✓ Avoid Rewriting Code

✓ Faster Development

3. SQUARE ROOT

Problem

Example

Result

5

Function

math.sqrt()

Example

math.sqrt(25)

Output

5.0

Important Note

sqrt() always returns a floating-point value.

6. CEIL FUNCTION

What Is Ceil?

Ceil rounds a decimal number upward to the next integer.

Function

math.ceil()

Example

math.ceil(4.3)

Output

5

More Examples

4.1 → 5

6.2 → 7

9.9 → 10

7. FLOOR FUNCTION

Floor rounds a decimal number downward to the nearest integer.
