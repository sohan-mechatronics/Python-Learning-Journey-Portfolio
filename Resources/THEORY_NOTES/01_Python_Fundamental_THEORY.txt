9. ARITHMETIC OPERATORS

Why?

Programs often perform calculations.

Assume

a = 20

b = 10

Addition = +

Example = a + b

Result = 30

Subtraction = -

Example = a - b

Result = 10

Multiplication = *

Example = a * b

Result = 200

Division = /

Example = a / b

Result = 2.0

Modulus = %

Example = a % b

Result = 0

Power = **

Example = a ** b

Result = 10240000000000

Floor Division = //

Example = a // b

Result = 2

Applications

✓ Calculators

✓ Engineering Calculations

✓ Financial Software

✓ Data Analysis

Why?

Programs need to compare values.

Assume

a = 20

b = 10

Greater Than = >

Output = True

Less Than = <

Output = False

Equal To = ==

Output = False

Not Equal To = !=

Output = True

Greater Than Or Equal To = >=

Output = True

Less Than Or Equal To = <=

Output = False

13. VARIABLE VS OUTPUT

Variable

Purpose = Store Data

Example

result = num1 - num2

Meaning

Value stored in memory.

Output

Purpose = Display Data

Example

print(result)

Meaning

Value shown on screen.

Memory Rule

= = Store Data

print() = Display Data
