1. STRINGS

What Is A String?

A String is a collection of characters stored together as text.

Examples

name = "SOHAN"

city = "Katihar"

branch = "Mechatronics"

Why Is It Important?

Because almost every software application uses text.

Real-Life Uses

✓ Login Systems

✓ Chat Applications

✓ Search Engines

✓ Websites

✓ Mobile Apps

✓ File Processing

✓ AI Applications

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

2. STRING CREATION

Why?

To store textual information.

Example

name = "SOHAN"

print(name)

Important Point

Strings are written inside quotation marks.

Examples

"Python"

"Engineering"

"Katihar"

3. STRING INDEXING ⭐

What Is Indexing?

Every character inside a string has a position.

That position is called an Index.

Example

String

SOHAN

Index

0 1 2 3 4

Character

S O H A N

Accessing Characters

name[0]

Output

S

name[2]

Output

H

Why Is It Important?

Because indexing allows access to individual characters.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

4. NEGATIVE INDEXING

Problem

How can we access characters from the end?

Solution

Negative Indexing

Example

String

SOHAN

Index

-5 -4 -3 -2 -1

Character

 S  O  H  A  N

Examples

name[-1]

Output

N

name[-2]

Output

A

Golden Rule

Positive Index

= Left To Right

Starts From 0

Negative Index

= Right To Left

Starts From -1

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

7. STRING METHODS ⭐

What Are String Methods?

Built-in tools used to perform operations on strings.

Important Methods

upper()

= Convert text to uppercase

Example

"sohan"

↓

"SOHAN"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

lower()

= Convert text to lowercase

Example

"SOHAN"

↓

"sohan"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

replace()

= Replace characters or words

Example

SOHAN

Replace S with M

↓

MOHAN

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

find()

= Find position of a character

Example

SOHAN

Find H

↓

2

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

count()

= Count occurrences

Example

SOHAN SOHAN

Count S

↓

2

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

8. STRING SLICING ⭐

Problem

What if we need only part of a string?

Solution

Slicing

Syntax

string[start:end]

Example

SOHAN

name[0:3]

Output

SOH

Rule

Start Index

✓ Included

End Index

✗ Excluded

Example

name[1:4]

Characters

O H A

Output

OHA

Why Is It Important?

Used in

✓ Data Processing

✓ Username Extraction

✓ Text Manipulation

✓ File Handling

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

9. SHORTCUT SLICING

Beginning To Position

name[:3]

Output

SOH

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Position To End

name[2:]

Output

HAN

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Entire String

name[:]

Output

SOHAN

Benefits

✓ Faster Coding

✓ Cleaner Syntax

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

10. REVERSE STRING ⭐

Problem

How can we reverse text?

Solution

Negative Step Slicing

Example

name[::-1]

Original

SOHAN

↓

Reversed

NAHOS

Understanding

[:]

= Entire String

-1

= Move Backward

Why Is It Important?

Used in

✓ Palindrome Programs

✓ String Processing

✓ Interview Questions
