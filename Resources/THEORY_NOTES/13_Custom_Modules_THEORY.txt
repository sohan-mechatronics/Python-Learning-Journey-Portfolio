1. CUSTOM MODULES

What Is A Custom Module?

A Custom Module is a Python file created by the programmer that contains reusable functions, variables or classes.

Why Is It Important?

Because large applications cannot be managed efficiently inside a single file.

Real-Life Uses

✓ Utility Functions

✓ Calculator Systems

✓ Automation Projects

✓ Robotics Software

✓ Industrial Applications

✓ Large Python Projects

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

2. CREATING A MODULE

Why?

To store reusable code separately.

Example File

mymodule.py

Inside File

def greet():
    print("Welcome Sohan")

Result

The function becomes reusable across multiple programs.

Benefits

✓ Reusability

✓ Clean Code

✓ Better Structure

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

3. IMPORTING A MODULE

Why?

Python must load a module before using its functions.

Syntax

import mymodule

Usage

mymodule.greet()

Output

Welcome Sohan

Benefits

✓ Access Reusable Code

✓ Reduce Repetition

✓ Organized Programming

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

4. MULTIPLE FUNCTIONS INSIDE A MODULE

Why?

A module can store related functions together.

Example

greet() = Welcome Message

add() = Addition Function

square() = Square Calculation

Usage

import mymodule

mymodule.greet()

mymodule.add()

mymodule.square()

Output

Welcome Sohan

30

25

Benefits

✓ One Module

✓ Multiple Functions

✓ Better Code Management

5. DIRECT FUNCTION IMPORT

Why?

Writing module names repeatedly increases code length.

Solution

Import only the required function.

Syntax

from mymodule import greet

Usage

greet()

Output

Welcome Sohan

Benefits

✓ Shorter Code

✓ Faster Development

✓ Cleaner Programs

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

6. MULTIPLE FUNCTION IMPORT

Why?

Sometimes multiple functions are required together.

Syntax

from mymodule import greet, add, square

Usage
