1. PANDAS

What Is Pandas?

Pandas is a Python library used for data handling, data analysis and data processing.

Why Is It Important?

Because most Data Science, Machine Learning and AI projects use Pandas.

Real-Life Uses

✓ Excel Data

✓ CSV Files

✓ Business Reports

✓ Employee Records

✓ Robotics Sensor Data

✓ Financial Analysis

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

2. IMPORTING PANDAS

Why?

Before using Pandas, Python must load the library.

Syntax

import pandas as pd

Understanding

pandas = Library

pd = Alias (Short Name)

Industry Standard

3. DATAFRAME

Why Do We Need DataFrames?

Suppose we have:

Name     Age

Sohan    20

Rahul    21

Amit     22

Using separate variables becomes difficult.

↓

Solution

DataFrame

Definition

A DataFrame is a table-like structure consisting of rows and columns.

Features

✓ Tabular Format

✓ Organized Data

✓ Easy Analysis

✓ Similar To Excel

✓ Row & Column Based

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

4. CREATING A DATAFRAME

Example

data = {
"Name":["Sohan","Rahul","Amit"],
"Age":[20,21,22]
}

df = pd.DataFrame(data)

Output

Name     Age

Sohan    20

Rahul    21

Amit     22

Result

Structured data created successfully.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

5. ACCESSING COLUMNS

Why?

Sometimes we need only specific information.

Example

Name Column

df["Name"]

Output

Sohan

Rahul

Amit

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Multiple Columns

df[["Name","Age"]]

Output

Name     Age

Sohan    20

Rahul    21

Amit     22

Important Concept

Single [ ]

One Column

Double [[ ]]

Multiple Columns

14. SORTING DATA

Why?

To arrange data systematically.

Ascending

Small → Large

df.sort_values("Age")

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Descending

Large → Small

df.sort_values(
"Age",
ascending=False
)

Benefits

✓ Organized Reports

✓ Easy Comparison

✓ Better Understanding
